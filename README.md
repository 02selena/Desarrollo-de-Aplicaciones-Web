# Galería Interactiva con JavaScript

Proyecto desarrollado para aplicar los fundamentos de JavaScript, manipulación del DOM y manejo de eventos.

## Funcionalidades
- Agregar imágenes mediante URL
- Seleccionar imágenes con un clic
- Eliminar la imagen seleccionada
- Interacción con teclado (Enter)

## Tecnologías Utilizadas
- HTML5
- CSS3 (Grid y animaciones)
- JavaScript (DOM y eventos)

## Publicación
El proyecto puede publicarse usando **GitHub Pages** desde la rama `main`.
